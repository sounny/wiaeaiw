## 🏗️ Project Components & Website Structure

### 1. Landing Portal (`index.html`)
A "wow" introductory page that sets the stage for the workshop. It should visualize the transition from traditional economic models to an AI-driven "Skills in Motion" paradigm.

### 2. Integrated Presentation (`presentation.html`)
A slide-based interface (custom HTML/JS) to be used as the primary lecture tool on May 6th.
*   **Content:** $2 Trillion AI Investment (2026), Gender Bias stats (37% female adoption gap), and the definition of the "Agentic Paradigm."
*   **Features:** Navigation keys, progress bar, high-resolution HSL gradients.

### 3. Technical Setup Guide (`setup.html`)
Concise instructions for participants to prepare their environments BEFORE the session.
*   **Requirements:** VS Code and the **Gemini CLI** (`npm install -g @google/gemini-cli`).
*   **Authentication:** Clear guide on obtaining and setting up Gemini API keys.

### 4. Interactive Lab Dashboard (`lab.html`)
The "follow-along" workspace for the second half of the workshop.
*   **Content:** Pre-written terminal commands for participants to copy and run in their VS Code terminal.
*   **Workflow:** Basic Query -> Multi-step Agent Task (e.g., "Analyze this folder and write a report").
*   **Focus:** Demonstrating AI as a "doer" rather than just a "talker."
