# WIA-Europe AI-W Workshop

**The Agentic Shift: Skills in Motion**

Official workshop portal for the [WIA-Europe Symposium 2026](https://www.wia-europe.org/wia-europe-symposium/).

## Overview

A 1-hour interactive workshop designed for the WIA-Europe Strasbourg Regional Network, exploring AI agency, gender equity in tech, and hands-on agentic architectures with Gemini CLI.

**Date:** May 6, 2026 | 13:00 CET | Online (Zoom)

## Structure

| Module | Page | Description |
|--------|------|-------------|
| 01 | `setup.html` | Pre-workshop: install VS Code, Node.js, and Gemini CLI |
| 02 | `presentation.html` | Slide deck: $4.4T AI economy, parity pivot, agentic paradigm |
| 03 | `lab.html` | Hands-on exercises: explain, automate, and refactor with AI |
| 04 | `lab.html#challenge` | Open challenge: design your own agentic workflow |
