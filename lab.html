Interactive Lab

Hands-on Lab

Follow along step by step. Each task builds on the previous one, progressing from simple queries to full agentic workflows.

Task 1

The Agentic Workspace

5 min

Before asking the AI to "do" things, we must build a persistent workspace. This structure allows the AI to understand its role, the project rules, and its memory across different sessions.

📁 Create your Project Folder

Terminal
mkdir my-agentic-app cd my-agentic-app mkdir .gemini touch agents.md code .

Open this folder in VS Code. Notice the agents.md file. This will be the "Brain" of your project.

💡
Persistence: By putting instructions in agents.md, we ensure that every time we launch the AI in this folder, it automatically adopts the personality and rules we've defined.
Task 2

Operating Directives

10 min

AI agents are most effective when they have a clear "Constitution." Let's define one. Open agents.md and paste the following:

Markdown (agents.md)
# Agent Identity & Directives ## 1. Primary Role I am a Senior Web Developer assisting with the WIA-Europe Symposium website. ## 2. Technical Rules - Use only Vanilla HTML and Modern CSS (no frameworks). - Use vibrant gradients and glassmorphism in designs. - Ensure all images have descriptive alt tags. ## 3. Communication Style - Be direct, technical, and professional. - Use arrow symbols (→) in call-to-action buttons.

🤖 Testing the Memory

Launch the Gemini CLI and ask it to describe its role. It will read the agents.md file and answer accordingly.

Gemini Prompt
Who are you and what are your rules for this project?
✅
Checkpoint: The AI should confirm it is a "Senior Web Developer" and list the rules you just wrote in the file. It is now "anchored" to your workspace.
Task 3

The AI Writes Its Own Brain

15 min

Instead of us writing the rules, let's have the AI analyze our project and write its own operating directives. This ensures the AI always understands the state of the codebase.

🧠 Generate gemini.md

Ask the AI to create a specialized instruction file for a specific feature.

Gemini Prompt
Analyze my project and create a `gemini.md` file in the root. In this file, define a "Feature Roadmap" for building a landing page that respects the rules in `agents.md`.
💡
Why this works: By having the AI write .md documentation for itself, we create a feedback loop. The AI documents what it understands, and its next responses are based on that documentation.

🚀 Execute the Roadmap

Now, tell it to follow the roadmap it just wrote:

Based on the roadmap in gemini.md, build the first section of the landing page.
Task 4

Iterating on Every Prompt

10 min

The final step is context management. When projects get large, the AI needs to know how to navigate folders. This is the difference between a chatbot and a true agent.

Gemini Prompt
Update `agents.md` to include a "Project Structure" section. List all current files and describe their purpose. Then, add a new rule: "Always check agents.md before proposing any code change."

Now, try to break a rule. Ask it to "Use Tailwind CSS."

⚠️
Observe: If the agents.md is working, the AI should refuse or warn you that Tailwind is against the "Vanilla CSS" rule you established in Task 2.
✅
Final Checkpoint: You have built an agentic environment where the AI follows persistent rules, documents its own progress, and maintains a consistent project architecture.
Challenge

Design Your Own Workflow

Now it is your turn. Using what you have learned, design and execute your own agentic workflow. Here are some ideas:

🌐

Data Dashboard

Ask the agent to create an interactive data dashboard that visualizes climate or space data.

📝

Research Summary

Ask the agent to analyze a research paper (paste text) and generate an executive summary with key takeaways.

🤖

Automation Script

Ask the agent to write a Python script that automates a tedious task in your work (e.g., renaming files, parsing data).

🏆
Pro Tip: The more specific and structured your prompt, the better the result. Frame your request like a project brief: define the goal, the deliverables, and any constraints.

Congratulations!

You have completed the AI Workshop: Skills in Motion. You now understand the agentic paradigm and have hands-on experience directing AI agents.
