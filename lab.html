Interactive Lab

Hands-on Lab

Follow along step by step. Each task builds on the previous one, progressing from simple queries to full agentic workflows.

Lab A

Explain & Verify

10 min

The first rule of Agentic AI: Trust, but Verify. Before letting an AI write code, we use it to explain and audit existing logic. This eliminates the "Black Box" effect and builds technical confidence.

🔍 Step 1: Human-in-the-Loop Audit

Paste a complex snippet of code into your terminal gemini prompt and ask it to explain it step-by-step.

Gemini Prompt
Explicate this code chunk and identify potential edge cases: [Paste your index.html here]
💡
Pedagogical Goal: This shift moves the participant from a "consumer" to an "auditor" of AI code, a key skill for technical leadership.
Lab B

Automate & Dry Run

15 min

Automation doesn't mean giving up control. We will now use the agent to automate a "boring" task (like adding meta tags or renaming files) using a Dry Run approach.

⚙ Step 2: Safe Automation

Draft a prompt that asks the AI to propose a change before applying it.

Gemini Prompt
Search for all `.html` files in my directory. Propose a single SEO meta description for each file based on its content. DO NOT modify the files yet — just output the proposed text for my review.
✅
Control Point: Only after you review the output and say "Proceed," should you allow the agent to edit the files. This is Agentic Accountability in action.
Lab C

Agentic Refactor

20 min

In the final lab, we use the agent to architect a roadmap for a complex feature. This demonstrates how AI handles non-linear, multi-step engineering tasks.

🧠 Step 3: Generating the Brain

Gemini Prompt
Analyze my `style.css`. Create a `responsive_roadmap.md` file that lists 5 key improvements to make this site perfect for mobile devices. For each improvement, explain WHY it is necessary for user experience.
🚀
Autonomous Execution: Once the roadmap is created, you can tell the AI: "Execute step 1 of the roadmap." It now has a persistent plan to follow.
Challenge

Design Your Own Workflow

Now it is your turn. Using what you have learned, design and execute your own agentic workflow. Here are some ideas:

🌐

Data Dashboard

Ask the agent to create an interactive data dashboard that visualizes climate or space data.

📝

Research Summary

Ask the agent to analyze a research paper (paste text) and generate an executive summary with key takeaways.

🤖

Automation Script

Ask the agent to write a Python script that automates a tedious task in your work (e.g., renaming files, parsing data).

🏆
Pro Tip: The more specific and structured your prompt, the better the result. Frame your request like a project brief: define the goal, the deliverables, and any constraints.

Congratulations!

You have completed the AI Workshop: Skills in Motion. You now understand the agentic paradigm and have hands-on experience directing AI agents.
